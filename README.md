# このリポジトリについて

このリポジトリは、`onlyone-restaurant.com`のBackend For Frontend(BFF)で〜す！！

# 環境構築
```
bun install
bun run dev
```

# 使用技術
- Hono
- Bun
- TypeScript
- Cloudflare Workers
